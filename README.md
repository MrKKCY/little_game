## 背景介绍
末日爆发了，你是末世的一个普通人，你要通过自己的判断和选择去决定你接下来走的路，必要的时候还需要一点点的运气。<br>
在末日击杀丧尸，升级自我，更好的活下去。

## 目前更新
- 商人系统，可以通过和商人交易提升血量和攻击力
- 暴击，和丧尸战斗的时候有概率触发暴击打出更高的伤害
- 更高的难度，没错，前面的梗图已经说了
- 先攻！你和丧尸会以**五五开**的概率随机决定先手，明显的，先手的一方会占有更大的优势
